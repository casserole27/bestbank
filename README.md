### The challenge

Requirements:
- build it from scratch
- follow the design
[Figma design](https://www.figma.com/file/5OS7V37a4fo3WIDLL7YisW/BestBank-(Copy)?node-id=0-1&t=YMvZxkwdgvRu8CfE-0)
